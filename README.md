Hinaan Simulator 🚀

Ekstensi VS Code esensial untuk menguji kekuatan mental dan kesabaran Anda saat koding. Menampilkan pesan interaktif yang penuh motivasi terbalik (hinaan) setiap kali Anda mendapati eror atau situasi tertentu di lingkungan kerja koding Anda.

> ⚠️ **DISCLAIMER / CATATAN PENTING:** 
> Ekstensi ini **hanya candaan semata yang lahir dari kegabutan seseorang**. Tidak ada maksud untuk benar-benar merendahkan atau menyinggung siapa pun. Gunakan dengan bijak dan jangan dimasukkan ke dalam hati!

🛠️ Panduan Pemasangan (Instalasi)

Karena ekstensi ini didistribusikan secara mandiri dalam format berkas `.vsix`, berikut adalah langkah mudah untuk memasangnya di VS Code Anda:

1. Buka halaman repositori GitHub ini, lalu masuk ke bagian **Releases** di sebelah kanan halaman.
2. Unduh berkas rilisan terbaru yang bernama **`hinaan-simulator-0.0.1.vsix`**.
3. Buka aplikasi **Visual Studio Code** Anda.
4. Masuk ke panel **Extensions** dengan menekan tombol kombinasi `Ctrl + Shift + X` (atau klik ikon kotak-kotak di bilah sisi kiri).
5. Klik ikon **titik tiga (`...`)** di pojok kanan atas panel Extensions tersebut.
6. Pilih opsi **"Install from VSIX..."** dari menu lungsur (*dropdown*).
7. Cari dan pilih berkas `hinaan-simulator-0.0.1.vsix` yang baru saja Anda unduh, lalu klik **Install**.
8. Tunggu beberapa saat hingga muncul notifikasi sukses di pojok kanan bawah. Ekstensi siap digunakan!

💻 Cara Pakai

Setelah berhasil terpasang, ekstensi ini akan bekerja secara otomatis di latar belakang. Anda tidak perlu melakukan konfigurasi tambahan. 

* **Deteksi Otomatis:** Setiap kali Anda menulis kode yang memicu eror di terminal, melakukan eksekusi yang salah, atau kodingan Anda tidak mau berjalan (*build fail*), simulator ini akan otomatis melemparkan pesan interaktif acak langsung ke layar Anda.
* **Uji Mental:** Hadapi setiap pesan dengan kepala dingin. Jadikan dorongan emosional ini sebagai bahan bakar untuk memperbaiki *bug* kode Anda dengan lebih cepat!
